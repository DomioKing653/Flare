# Flare

This main git repo for flare programming language.

---

## Why to chose flare
- **Staticly typed**:catch all of the type errors at compile time
- **Fast compilation speed**:compile your programs in seconds

---

## Staring with flare

For now flare can be only build from source. For more info look at the **[Building from source](docs/BUILDING.md)**.

When you are ready you may look at the [Getting started](docs/GETTING_STARTED.md)
